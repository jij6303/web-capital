### #1 세계 수도 퀴즈 웹 게임 구현 (2026-04-05)
**왜 이 작업을 했나**
> 프로젝트의 핵심 기능인 수도 퀴즈 게임의 기반을 만들기 위해

**뭘 했나**
> `app.js` 138줄: 퀴즈 상태 관리, 문제 로딩, 보기 셔플, 정답 체크, 점수 계산
> `data.js` 195줄: ~200개국 국가/수도 데이터, 지역 메타데이터
> `index.html` 241줄: 퀴즈 화면, 보기 컨테이너, 피드백 영역, 결과 화면 구조
> `style.css` 262줄: 다크 테마(#0f172a, #1e293b), 그리드 레이아웃, 버튼 상태 스타일

**어떤 구조를 선택했고 왜**
> SPA(Single Page Application) — 화면 전환 시 페이지 리로드 없이 상태 유지 가능.
> 데이터(data.js), 로직(app.js), 마크업(index.html), 스타일(style.css) 4파일 역할 분리.
> 오답 보기는 전체 나라 풀에서 뽑아 지역 제한 없이 난이도 다양성 확보.

**배운 것 / 몰랐던 것**
> `querySelectorAll` 다중 선택자로 정답/오답 버튼 일괄 스타일링 가능.
> 한국어 charset 호환성을 위해 HTML `<meta charset="UTF-8">` 필수.

**에러 & 해결**
> 에러: 지역 필터링 미구현 (feat #3에서 해결)
> 원인: 초기 버전은 전체 국가 풀에서만 출제
> 해결: #3에서 지역 선택 화면 + 필터링 로직 추가

### #1 수도 및 국가명 한국어 번역 (2026-04-05)
**왜 이 작업을 했나**
> 한국어 사용자를 위한 완전한 로컬라이제이션 — 영어 데이터로는 UX가 떨어지기 때문

**뭘 했나**
> `data.js` 전체 195개 국가명/수도명을 한국어로 번역
> ISO 국가 코드(alpha-2)는 지도 연동을 위해 영어 유지
> 예: "Afghanistan" → "아프가니스탄", "Kabul" → "카불"

**어떤 구조를 선택했고 왜**
> 데이터 스키마 구조는 그대로 유지하고 값만 교체 — 로직 변경 없이 번역 완료.
> 단일 커밋으로 전체 번역 완료해 중간 상태(일부 영어/일부 한국어) 방지.

**배운 것 / 몰랐던 것**
> 긴 한국어 수도명(예: "스리랑카의 제임스 지야와르데네푸라코테")이 UI 레이아웃에 영향을 줄 수 있음.
> 국가 표준 한국어 표기 확인 필요 (터키 → 튀르키예 등).

**에러 & 해결**
> 에러: 일부 국가 번역 정확도 미검증
> 원인: 드문 국가는 표준 한국어 표기 기준이 불명확
> 해결: 추후 교육부/외교부 표기 기준 대조 필요

### #3 지역 챕터 선택 화면 추가 (2026-04-05)
**왜 이 작업을 했나**
> 전체 216개국 랜덤 퀴즈는 학습 효율이 낮아, 지역별로 집중 학습할 수 있도록 개선하기 위해

**뭘 했나**
> `data.js` 418줄: 7개 지역 정의(`REGIONS` 배열), 국가 수 216개로 확장, 중복 제거 로직 추가
> `index.html`: `#select-screen` 지역 카드 레이아웃 추가, D3.js + TopoJSON CDN 연동, 퀴즈 로직을 inline script로 이전
> `style.css`: `.region-card` 그리드 스타일, hover 효과, 모바일 미디어 쿼리 추가

**어떤 구조를 선택했고 왜**
> 퀴즈 로직을 `app.js`에서 `index.html` inline script로 이전 — 지역 선택 상태를 단일 스코프에서 관리하기 위해.
> D3.js로 SVG 세계 지도 렌더링 — TopoJSON의 수치 코드를 ISO alpha-2로 변환하는 `NUM_TO_A2` 매핑 객체 사용.
> `mapReady` 플래그 + `pendingHighlight` 큐로 지도 비동기 로딩 레이스 컨디션 해결.
> Set 기반 중복 제거로 데이터 확장 중 발생한 코모로 중복 항목 제거.

**배운 것 / 몰랐던 것**
> TopoJSON은 UN M49 수치 코드를 사용해 ISO alpha-2 변환 매핑 필요.
> D3.js 지도 렌더링은 비동기 — 로딩 전 요청된 하이라이트는 큐에 보관 후 처리.
> 지역 필터링으로 학습 집중도가 크게 향상됨.

**에러 & 해결**
> 에러: 지역별 국가 수 합산이 전체와 불일치
> 원인: 일부 국가 누락
> 해결: 누락 국가 추가해 216개로 확장
>
> 에러: SVG 지도에서 일부 국가 하이라이트 실패
> 원인: TopoJSON 포맷마다 ID 속성명이 다름 (`id`, `data-id` 등)
> 해결: 다중 선택자 fallback으로 여러 포맷 대응
